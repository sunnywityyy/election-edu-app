# ElectionCheck - AI-Powered Indian Election Myth Buster

## Challenge Vertical
Election Process Education

## Approach
Built an AI-powered fact-checker using Google Antigravity and Groq API that:
- Accepts any claim about Indian elections
- Instantly fact-checks using AI (TRUE/FALSE/MISLEADING)
- Shows confidence scores and official sources
- Supports English, Hindi, and Telugu

## How It Works
1. User types any election-related claim
2. AI analyzes the claim against official Election Commission of India data
3. Returns structured verdict with explanation and source
4. Results are shareable as image cards

## Features
- AI Fact Checker with confidence scoring
- 6-step visual guide of Indian election process
- Multilingual support (English/हिंदी/తెలుగు)
- Shareable fact-check cards
- Mobile-responsive design

## Tech Stack
- React + Vite
- Tailwind CSS
- Groq AI API
- Built with Google Antigravity
